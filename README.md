# NIS process: исключение из реестра после увольнения (Camunda 8.8 / BPMN 2.0)

## Описание кейса
Кейс: военнослужащий уволен из ВС РФ. Ответственный за НИС в воинской части не выполнил необходимые мероприятия, пока человек был на службе.  
Цель: отрисовать схему корректного процесса

## Нормативная база (использовалась при построении процесса)
- Федеральный закон №117-ФЗ (НИС)
- Постановление Правительства РФ №89 (Правила функционирования НИС/реестра)
- Приказ Министра обороны РФ №477 (порядок действий/оформления списков и взаимодействия)
- Постановление Правительства РФ №655
- Постановление Правительства РФ №370

- ## Содержимое репозитория
- `Кейс.bpmn` — основная диаграмма процесса
- `forms` — формы для user tasks
- `docs` —  нормативные документы
